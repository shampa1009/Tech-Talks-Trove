🚨 “Vibe Coding” এর অনেক Side Effect আছে

AI দিয়ে অ্যাপ বানানো এখন এত সহজ হয়ে গেছে যে-
অনেকেই software architecture, security, hosting, database, load balancing, scaling কিছু না জেনেই production app launch করে দিচ্ছে.

শুরুতে এটা magical লাগে. কিন্তু সমস্যা শুরু হয় যখন:
👉 Real users আসে
👉 Traffic বাড়ে
👉 Data leak হয় 
👉 App crash করে
👉 Store policy violation ধরা পড়ে

সম্প্রতি Apple App & Google Play Store -এ অনেক low-quality ও risky AI-generated app নিয়ে কড়াকড়ি শুরু হয়েছে.

একটা interesting real-world case👇
Replit-এর AI vibe coding platform নিয়ে App Store-এ সমস্যা হয়. Apple তাদের update block করে দেয় এবং আরো কিছু AI-generated app remove/hold করা হয়. কারণ:
• Dynamically generated code
• Security concern
• Policy violation
• Unstable functionality

Replit is a browser-based development platform where the user can:
• Write code
• Run code
• Deploy apps
👉 All from a browser — no local setup needed
It was founded with the vision:
“Anyone should be able to build software from anywhere.”

Recent security research-এ দেখা গেছে হাজার হাজার AI-generated app public internet-এ sensitive data expose করছে:
• Medical data
• Personally indentifiable data 
• Financial documents
• Internal business files

এইক্ষেত্রে সমস্যা AI না. সমস্যা হলো:
People are shipping software without understanding software.

Software development শুধু UI বানানো না. এর মধ্যে আছে:
• Architecture
• Scalability
• Security
• Infra
• Monitoring
• Optimization
• Maintenance

AI prototype বানাতে পারে.
কিন্তু production-grade system এখনো engineering mindset চায়.

Prompt can generate code. But engineering builds systems.
